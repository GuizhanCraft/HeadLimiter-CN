# HeadLimiter 头颅限制

该插件仅在 Paper 中才能运行。也许在Paper的fork版本中也能正常运行，但没有经过测测试。

该插件会限制玩家在一个区块中可放置的货运节点的数量，默认为25个

## 下载 HeadLimiter
你可以在这里下载 HeadLimiter: [点此下载](https://github.com/ybw0014/HeadLimiter-CN/actions/workflows/build.yml)
